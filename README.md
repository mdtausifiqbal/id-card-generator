# ID Card Generator

This app allows service franchisee of Voltas, Carrier Midea and Daikin to create ID Card of their employee or staff.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
